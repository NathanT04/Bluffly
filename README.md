# Bluffly

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 20.3.6.

## 1. Install Dependencies
**Backend**
```cd backend
npm install
```

**Frontend**
```cd frontend
npm install
```

## 2. Deploy
**Run Both Servers in Seperate Terminals**
**Backend**
```
cd backend
node app.js
npm start
```

**Frontend**
```
cd frontend
ng serve --proxy-config proxy.conf.json
```
